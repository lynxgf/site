# История разработки интернет-магазина "Матрасовъ"

## Этап 1: Создание основы сайта и базы данных
- Разработана структура базы данных с таблицами для пользователей, товаров, корзины, заказов
- Создан современный премиальный дизайн сайта, соответствующий бренду "Матрасовъ"
- Настроен базовый функционал интернет-магазина: каталог товаров, страницы продуктов

## Этап 2: Разработка системы настройки товаров
- Реализована система конфигурации товаров с выбором размеров (односпальные, двуспальные и т.д.)
- Добавлена возможность выбора категорий тканей с учетом их влияния на стоимость
- Интегрирована опция выбора конкретных тканей с предпросмотром
- Внедрена возможность указания кастомных размеров для некоторых товаров
- Добавлен параметр подъемного механизма для кроватей с соответствующей наценкой

## Этап 3: Корзина и процесс оформления заказа
- Создана функциональная корзина с возможностью изменения количества товаров
- Внедрено динамическое обновление итоговой стоимости
- Разработан многошаговый процесс оформления заказа
- Добавлены формы для ввода контактной информации и адреса доставки
- Реализована возможность выбора способа доставки (курьером за 500₽ или самовывоз бесплатно)
- Добавлен выбор метода оплаты (наличные или банковская карта)

## Этап 4: Административная панель
- Создана защищенная административная панель с аутентификацией
- Разработан интерфейс управления товарами (добавление, редактирование, удаление)
- Добавлена система управления пользователями
- Реализована страница для управления заказами с возможностью изменения их статуса
- Интегрирована функциональность для просмотра подробной информации о каждом заказе

## Этап 5: Улучшение фильтрации и поиска
- Добавлена фильтрация товаров по категориям (кровати, матрасы)
- Реализован поиск по названию товара
- Внедрена сортировка по цене и популярности

## Этап 6: Оптимизация расчета стоимости заказа
- Исправлен расчет итоговой стоимости заказа с учетом способа доставки
- Доработана система для правильного учета стоимости доставки (500₽ для курьерской доставки, 0₽ для самовывоза)
- Оптимизирована логика расчета общей суммы заказа
- Улучшен интерфейс отображения цен на странице оформления заказа

## Этап 7: Усовершенствование административной панели заказов
- Добавлена фильтрация заказов по статусу (ожидает оплаты, в обработке, отправлен, доставлен, завершен, отменен)
- Реализована фильтрация заказов по способу доставки (курьером, самовывоз)
- Внедрены визуальные индикаторы метода доставки для быстрого восприятия информации
- Добавлены информативные значки для различных статусов заказов
- Улучшен интерфейс таблицы заказов для более эффективного управления

На каждом этапе особое внимание уделялось премиальному дизайну сайта, адаптивности под различные устройства и удобству использования как для клиентов, так и для администраторов магазина.

Сайт полностью функционален и готов к использованию. Клиенты могут просматривать каталог, настраивать товары под свои потребности и успешно оформлять заказы, а администраторы – эффективно управлять всеми аспектами интернет-магазина.
